# Expression

The core feature of this language is `Expression`.

**Syntax**

```
<expr>
```


## What is expression?

If source code can `evaluate` immediately and can return the `result`.
That is an expression.



**Example**

```lisp
"Hey there" 
123
```

## String

```lisp
"Hello World"
```

## Number

```lisp
123
456
```
